Install

Choose a preview install path.

Homebrew is the simplest path. The one-line installer can be pinned to a checksum.

Homebrew cask
brew install --cask https://raw.githubusercontent.com/ShunmeiCho/cmd-ime/main/Casks/cmd-ime.rb
Good default for repeatable local installs.
Pinned installer
CMDIME_VERSION=0.1.13 CMDIME_SHA256=86fb954c... /bin/bash -c "$(curl -fsSL ...)"
Use the published checksum when you install directly.
